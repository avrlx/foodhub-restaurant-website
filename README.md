# Foodhub Restaurant Website

A responsive restaurant landing page built with HTML, CSS, and JavaScript. The site presents a modern food brand experience with a sticky navigation bar, product cards, testimonials, and a footer with contact details.

## Features

- Responsive one-page layout
- Mobile menu toggle and shopping cart drawer
- Smooth in-page scrolling for section links
- Active section highlighting in the navigation
- Updated footer with quick links, contact details, and dynamic year
- Animated hero and section visuals

## Project Structure

```text
index.html
assets/
  css/
    foodhub.css
    media_query.css
  images/
  js/
    foodhub.js
```

## How to Run

1. Open `index.html` in a browser.
2. Or use a local development server for a better preview experience.

Example with VS Code Live Server:

- Install the Live Server extension
- Right-click `index.html`
- Choose Open with Live Server

## Notes

- The page uses Ionicons from a CDN for the icon set.
- The layout is designed to work across mobile, tablet, and desktop breakpoints.
